⚛️vue-node-Chatroom
---

这是一个基于nodejs+express+socket.io模块构建的聊天系统

前端使用Vue作为前端框架。服务端使用express搭建web服务，使用socket.io来创建服务端socket服务，可以进行匿名聊天，多人聊天，群组聊天。

## 使用方式

# 克隆到本地
git clone https://github.com/Oudanlu/node-Chatroom.git

# 进入文件夹
cd node-Chatroom

# 安装依赖
npm install 或 yarn(推荐)

# 启动服务
npm run dev
